# Smart Public Complaint App

Smart Public Complaint App एक नागरिक शिकायत पोर्टल है जहा यूजर अपनी शिकायत दर्ज कर सकता है, उसका स्टेटस देख सकता है और ट्रैक कर सकता है। प्रशासनिक और पुलिस ऑथोरिटी के लिए अलग डैशबोर्ड भी मौजूद है।

## Features
- User authentication
- Register and submit complaints
- Track complaint status
- Admin dashboard for complaint management
- Police dashboard for challan management

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Auth, Firestore
